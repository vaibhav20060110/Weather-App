# 📖 About

The Weather App is a modern weather forecasting application that allows users to search for cities around the world and instantly view current weather conditions.

The application displays essential weather information including temperature, humidity, wind speed, pressure, visibility, and weather conditions. It also includes an interactive weather map and weather comparison functionality for a better user experience.

Designed with a clean and responsive interface, the application works smoothly across desktops, tablets, and mobile devices.

---

# ✨ Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature
- ☁️ Live weather conditions
- 💧 Humidity
- 💨 Wind speed
- 🌬️ Atmospheric pressure
- 👀 Visibility information
- 🌅 Sunrise & Sunset timings
- 🗺️ Interactive weather map
- ⚖️ Weather comparison between cities
- 📱 Responsive design
- ⚡ Fast API-based weather updates
- 🎨 Clean and modern UI

### Clone the repository


